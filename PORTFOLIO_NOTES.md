# Portfolio Developer Notes

Tôi là Frontend Developer định hướng phát triển các ứng dụng web hiện đại và thân thiện với người dùng.

## Tôi làm được gì?

Tôi có thể xây dựng giao diện responsive, quản lý state, gọi API, xử lý form và phát triển ứng dụng React nhiều trang.

## Tôi sử dụng công nghệ gì?

HTML, CSS, JavaScript, TypeScript, React, Tailwind CSS, SCSS, Redux Toolkit, React Router, React Hook Form, Zod và Git.

Tôi đang tìm kiếm vị trí Frontend Developer Intern hoặc Fresher để phát triển kỹ năng và tham gia xây dựng các sản phẩm thực tế.
